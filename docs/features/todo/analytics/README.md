# Analytics Epic

## Overview

This epic contains admin analytics dashboard features for tracking student engagement and platform metrics.

## Features (8 total)

- **87**: Create analytics data aggregation functions
- **88**: Display student engagement metrics
- **89**: Display module completion rates
- **90**: Display time spent analytics
- **91**: Display drop-off points
- **92**: Display active vs inactive students
- **93**: Display revenue metrics
- **94**: Export analytics data

## Implementation Order

1. Start with feature 87 (data aggregation)
2. Build individual metric displays (88-93)
3. Add export functionality (94)

## Dependencies

- Requires Progress Tracking epic (features 48-55)
- Requires Admin role (feature 47)
- Requires Subscriptions (for revenue metrics)
