# Email Notifications Epic

## Overview

This epic contains features for sending email notifications to students for various events.

## Features (6 total)

- **101**: Set up email service integration
- **102**: Send welcome email on signup
- **103**: Send assignment due date reminders
- **104**: Send module completion celebration emails
- **105**: Send weekly digest emails
- **106**: User email notification preferences

## Implementation Order

1. Start with feature 101 (email service)
2. Build individual email types (102-105)
3. Add preferences (106)

## Dependencies

- Independent epic
- Can integrate with existing notification system
