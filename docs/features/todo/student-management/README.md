# Student Management Epic

## Overview

This epic contains admin features for managing individual students, viewing their progress, and performing bulk operations.

## Features (6 total)

- **95**: Admin can view individual student progress
- **96**: Admin can view student activity timeline
- **97**: Admin can send message to student
- **98**: Admin can export student data
- **99**: Admin can view student submissions summary
- **100**: Admin can view student quiz results summary

## Implementation Order

1. Start with feature 95 (view progress)
2. Build activity timeline (96)
3. Build messaging (97)
4. Build export (98)
5. Build summaries (99-100)

## Dependencies

- Requires Admin role (feature 47)
- Requires Progress Tracking (features 48-55)
- Requires Assignments (features 56-65)
- Requires Quizzes (features 66-73)
