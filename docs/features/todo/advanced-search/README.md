# Advanced Search Epic

## Overview

This epic contains features for searching across content, posts, members, and other platform data.

## Features (5 total)

- **107**: Create search index/data structure
- **108**: Search across content and modules
- **109**: Search across community posts
- **110**: Search across members
- **111**: Advanced search filters and saved searches

## Implementation Order

1. Start with feature 107 (search infrastructure)
2. Build content search (108)
3. Build post search (109)
4. Build member search (110)
5. Add filters and saved searches (111)

## Dependencies

- Independent epic
- Can use PostgreSQL full-text search or external service
