# Certificates Epic

## Overview

This epic contains features for generating and displaying completion certificates and achievement badges.

## Features (4 total)

- **74**: Create certificate table schema
- **75**: Generate certificate when module completed
- **76**: Display certificate to student
- **77**: Create achievement badges system

## Implementation Order

1. Start with feature 74 (certificate table)
2. Build certificate generation (75)
3. Build certificate display (76)
4. Add badges system (77)

## Dependencies

- Requires Progress Tracking epic (features 48-55) for completion tracking
