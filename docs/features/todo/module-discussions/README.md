# Module Discussions Epic

## Overview

This epic contains features for module-specific discussion forums where students can ask questions tied to content.

## Features (5 total)

- **82**: Create module_discussion table schema
- **83**: Student can create discussion post in module
- **84**: Student can reply to discussion
- **85**: Display discussions on module page
- **86**: Link discussions to specific content items

## Implementation Order

1. Start with feature 82 (discussion table)
2. Build discussion creation (83)
3. Build replies (84)
4. Build display (85)
5. Add content linking (86)

## Dependencies

- Requires Classroom epic (features 29-33) for modules
- Can reuse Community Posts patterns
